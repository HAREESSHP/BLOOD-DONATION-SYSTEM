Blood Type Compatibility

Understanding blood type compatibility is crucial for safe transfusions. See which blood types can donate to and receive from each other.

Blood Type Can Donate To Can Receive From
O- All Blood Types Universal Donor O-
O+ O+, A+, B+, AB+ O+, O-
A- A-, A+, AB-, AB+ A-, O-
A+ A+, AB+ A+, A-, O+, O-
B- B-, B+, AB-, AB+ B-, O-
B+ B+, AB+ B+, B-, O+, O-
AB- AB-, AB+ AB-, A-, B-, O-
AB+ AB+ All Blood Types Universal Recipient

O- is the Universal Donor

O negative blood can be given to anyone in emergencies when there's no time to check blood type.

AB+ is the Universal Recipient

AB positive patients can receive blood from any blood type, making them universal recipients.

Rh Factor Matters

Negative blood types can donate to positive, but positive cannot donate to negative Rh types.
